1. What is Web Technology?
    Web technology refers to the tools, protocols, and languages (like HTML, CSS, JavaScript, servers, databases) used to        build, run, and interact with websites and web applications.


2. Real-time examples of Web Technology
    -E-commerce: Online stores (Amazon, Shopify).
    -Social Media: Platforms for connection (Facebook, Twitter).
    -Content Delivery: Blogs, news sites, streaming (YouTube).
    -Online Learning (E-Learning): Platforms like Coursera, Udemy.


3. What is JavaScript?
    -JavaScript is the programming language of the web.
    -It can calculate, manipulate and validate data.
    -It can update and change both HTML and CSS.


4. Which file extension is used for JavaScript?
    .js file extension is used for JavaScript.


5. Why do we use Node.js?
    We use Node.js primarily to build fast, scalable network applications and to enable server-side development using JavaScript.
